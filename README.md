# API de Jogos de Video Game com CI/CD

Este projeto consiste em uma API REST desenvolvida em Java com Spring Boot 3 e Java 21, contendo uma esteira automatizada de CI/CD via GitHub Actions com deploy em uma instância EC2 na AWS usando Docker.

## 🚀 Como Acessar a API
A aplicação está em produção e pode ser acessada publicamente através do link abaixo:
* **Endpoint de Jogos:** `http://SEU_IP_PUBLICO_DA_EC2/api/jogos`

## 🛠️ Tecnologias Utilizadas
* **Backend:** Java 21, Spring Boot 3 (Spring Web, Spring Boot Test)
* **Build & Testes:** Maven
* **Containers:** Docker & Docker Hub
* **CI/CD:** GitHub Actions
* **Cloud:** AWS EC2 (VM Ubuntu)
